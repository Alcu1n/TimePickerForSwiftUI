# Folder: Sources/WheelPickerKit

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时,更新此文档。

This folder contains the shipped picker module and nothing from the demo screen. / 本目录只包含对外分发的选择器模块，不包含任何示例页面内容。  
It exports `TimerWheelPicker` while keeping wheel rendering and haptic services package-private. / 它导出 `TimerWheelPicker`，同时将滚轮渲染与触感服务保持为包内私有实现。  
The invariant is that package consumers get a clean picker view plus explicit styling/value APIs, not demo chrome. / 最重要的不变量是包使用者拿到的是干净的 picker 视图和显式样式/数值 API，而不是示例装饰层。

## Files
- `TimerWheelPicker.swift`: public component - Public timer-facing picker API and styling surface / 面向计时器的公开组件 API 与样式入口
- `WheelPickerView.swift`: package-private view - Arc wheel rendering, tick gradients, and selection syncing / 圆弧滚轮渲染、刻度渐变与选择同步
- `WheelHapticController.swift`: package-private service - Dedicated haptic and click-audio feedback engine / 专用震动与点击音反馈引擎
- `.folder.md`: local map - Folder responsibilities and file list / 目录职责与文件列表
