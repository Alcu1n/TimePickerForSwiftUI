# Folder: Sources/WheelPickerKit

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时,更新此文档。

This folder contains the shipped picker module and no demo-only chrome. / 本目录只包含对外分发的选择器模块，不包含任何示例专用装饰层。  
It exports `TimerWheelPicker` with immersive-by-default behavior, mirrored shallow-arc swipe semantics, readable style aliases, explicit initial-selection fallback, viewport tick-fade controls, and value-position knobs while keeping full-bleed arc geometry and low-latency metallic feedback package-private. / 它导出默认沉浸式行为、浅弧镜像滑动语义、易读样式别名、显式初始值回退、刻度视口褪色控制与数字位置参数的 `TimerWheelPicker`，同时将贴边圆弧几何与低延迟金属反馈保持为包内私有实现。  
The invariant is that consumers configure arc skins and first-load defaults through explicit APIs instead of importing app-layer layout. / 最重要的不变量是使用者只能通过显式 API 配置圆弧皮肤和首次加载默认值，而不是引入应用层布局。

## Files
- `TimerWheelPicker.swift`: public component - Public picker API, immersive default behavior, mirrored shallow-arc semantics, initial-selection fallback, arc profiles, tick-fade/value-position knobs, readable customization aliases, and reusable style presets / 公开选择器 API、默认沉浸式行为、浅弧镜像语义、初始值回退、圆弧轮廓、刻度褪色与数字位置参数、易读定制别名与可复用样式预设
- `WheelPickerView.swift`: package-private view - Shared full-bleed arc geometry, mirrored shallow-arc ordering, viewport-driven tick fading, hit-region tuning, and aggressively tightened label placement / 共享贴边圆弧几何、浅弧镜像排序、视口驱动的刻度褪色、命中区调优与大幅收紧的标签定位
- `WheelHapticController.swift`: package-private service - Dedicated rigid-impact plus low-latency metallic ratchet engine for per-tick wheel motion / 用于逐刻度滚轮运动的专用刚性冲击加低延迟金属棘轮引擎
- `.folder.md`: local map - Folder responsibilities and file list / 目录职责与文件列表
