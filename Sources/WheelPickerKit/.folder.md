# Folder: Sources/WheelPickerKit

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时,更新此文档。

This folder contains the shipped picker module and no demo-only chrome. / 本目录只包含对外分发的选择器模块，不包含任何示例专用装饰层。  
It exports `TimerWheelPicker` plus style presets while keeping arc geometry, tick masking, and haptics package-private. / 它导出 `TimerWheelPicker` 与样式预设，同时将圆弧几何、刻度遮罩与触感保持为包内私有实现。  
The invariant is that consumers configure arc skins through explicit style APIs instead of importing app-layer layout. / 最重要的不变量是使用者只能通过显式样式 API 配置圆弧皮肤，而不是引入应用层布局。

## Files
- `TimerWheelPicker.swift`: public component - Public picker API, arc profiles, and reusable style presets / 公开选择器 API、圆弧轮廓与可复用样式预设
- `WheelPickerView.swift`: package-private view - Shared arc geometry, tick masking, and selection syncing / 共享圆弧几何、刻度遮罩与选择同步
- `WheelHapticController.swift`: package-private service - Dedicated haptic and click-audio feedback engine / 专用震动与点击音反馈引擎
- `.folder.md`: local map - Folder responsibilities and file list / 目录职责与文件列表
