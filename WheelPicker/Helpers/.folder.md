# Folder: WheelPicker/Helpers

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时,更新此文档。

This folder provides reusable wheel rendering, interaction, and stable haptic-plus-audio feedback primitives. / 本目录提供可复用的滚轮渲染、交互与稳定的震动加音频反馈原语。  
It should not own product copy or timer-specific messaging from parent screens. / 它不应持有来自上层界面的业务文案或计时器专属语义。  
The invariant is that visual behavior stays generic while selection and multimodal feedback engines stay discrete and stable. / 最重要的不变量是视觉行为保持通用，而选择与多模态反馈引擎逻辑保持离散且稳定。

## Files
- `WheelPickerView.swift`: helper view - Arc wheel rendering, gradient ticks, haptics, and selection syncing / 圆弧滚轮渲染、渐变刻度、震动反馈与选择同步
- `WheelHapticController.swift`: helper service - Persistent Core Haptics engine for per-tick haptic plus click-audio feedback / 用于逐刻度震动加点击音反馈的持久化 Core Haptics 引擎
- `.folder.md`: local map - Folder responsibilities and file list / 目录职责与文件列表
