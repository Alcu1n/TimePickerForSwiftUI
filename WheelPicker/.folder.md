# Folder: WheelPicker

> Trigger: When this folder's structure/responsibilities/file list changes, update this document.
> 触发条件:当本文件夹的结构/职责/文件列表变化时,更新此文档。

This folder contains the app scene, premium timer screen, and tuning controls. / 本目录包含应用场景、高级计时器界面与调节控制。  
It receives reusable wheel primitives from `Helpers/` and exposes the runnable demo plus size tuning UI. / 它从 `Helpers/` 接收可复用滚轮原语，并向应用入口暴露可运行示例与尺寸调节界面。  
The invariant is that app-facing time semantics and tuning state live here, not inside low-level wheel rendering. / 最重要的不变量是面向应用的时间语义与调节状态必须留在这里，而不是下沉到低层滚轮渲染。

## Files
- `ContentView.swift`: screen - Timer picker demo composition, scaling, and tuning UI / 计时器选择器示例的组合、缩放与调节界面
- `WheelPickerApp.swift`: app root - Application entry scene / 应用入口场景
- `WheelPicker.entitlements`: config - App sandbox entitlements / 应用沙盒权限配置
- `Helpers/`: child folder - Reusable wheel rendering components / 可复用滚轮渲染组件
- `Assets.xcassets/`: child folder - App icons and color assets / 应用图标与颜色资源
- `.folder.md`: local map - Folder responsibilities and file list / 目录职责与文件列表
